ScoreKeeper

Capstone Project

Rachel Cavin

The Problem

  • Teachers use outdated technology, or no technology at all for grading, storing, and creating assignments.
  • This portion of a teachers' job robs time they could allocate to instructional design.
  • Research indicates of those who go into teaching, 40-50% are gone within 5 years. The burdens on teachers need to be addressed.

ScoreKeeper

  • Early stage product that aims to reduce the time teachers spend on assessment and scoring.
  • Gradebook functionality for managing grades, students, courses, and assignments.
  • Future development plans that include automatically graded multiple choice assessments, simplified rubric creation and grading, and better student progress monitoring systems.

User Stories

Teachers

As a teacher I want to be able to review scores so that I can view scores by student, assignment, and class.

As a teacher I want to be able to add, edit, and remove students, classes, and assignments.

As a teacher I want to be able to manually edit grades in the gradebook.

Students

As a student I want to be able to view my scores so that I can see my performance across all of my assignments.
